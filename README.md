# NEITS RMS

## NEITS Repair Management System

A modern cloud-based Repair Management System for computer, laptop, printer, and IT service centers.

## Features

- Customer Management
- Job Card Management
- Device Tracking
- Technician Assignment
- Repair Workflow
- Estimate Approval
- Billing & Invoice
- WhatsApp Notifications
- Google Sheets Synchronization
- AI Customer Assistant
- Inventory Management
- Dashboard & Reports

## Technology Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL

### Automation
- n8n

### Hosting
- Docker
- Nginx
- Ubuntu Server

## Developed For

**NEITS (Nepal Electronics & Information Technology Services)**

Kathmandu, Nepal

## Project Vision

To build a modern, AI-powered, cloud-based Repair Management System for computer and electronics service centers with complete workflow automation—from customer job entry to delivery, billing, WhatsApp notifications, inventory management, reporting, and AI-assisted customer support.
